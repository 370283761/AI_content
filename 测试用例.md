## 使用说明

- **TC 编号**：`TC-XXX` 格式，按功能模块递增。
- **状态**：✅ 通过 | ❌ 失败 | ⏸️ 暂停 | 🚧 待编写
- **Python 脚本**：对应的自动化测试脚本路径（如 `tests/api/test_project.py::test_create_project`）。
- **最后执行**：最近一次测试时间。

### TC-001: 创建项目

**前置条件**：无

**步骤**：
1. 访问 `http://localhost:3000/projects/new`
2. 输入项目名称："测试项目 001"
3. 选择内容来源："故事创意"

## Python 测试脚本示例

### 安装依赖

```bash
cd tests/api
pip install -r requirements.txt
```

### 运行测试

```bash
# 运行所有测试
pytest -v

# 运行单个文件
pytest test_project.py -v

# 运行单个用例
pytest test_project.py::test_create_project -v
```

### 示例脚本（test_project.py）

```python
import pytest
import requests

BASE_URL = "http://localhost:3000/api"

def test_create_project():
    """TC-A001: 创建项目"""
    payload = {
        "name": "测试项目 001",
        "sourceKind": "idea",
        "sourceContent": "一位少年在末日世界中寻找失散的妹妹",
        "params": {
            "style": "二次元赛璐璐",
            "aspectRatio": "9:16",
            "targetEpisodeLength": 60
        }
    }
    
    response = requests.post(f"{BASE_URL}/projects", json=payload)
    
    assert response.status_code == 200
    data = response.json()
    assert "id" in data
    assert data["name"] == "测试项目 001"
    
    # 清理
    project_id = data["id"]
    requests.delete(f"{BASE_URL}/projects/{project_id}")

def test_list_projects():
    """TC-A002: 获取项目列表"""
    response = requests.get(f"{BASE_URL}/projects")
    
    assert response.status_code == 200
    data = response.json()
    assert isinstance(data, list)
```
