## 使用说明

- **功能编号**：`FUNC-XXX` 格式，按交付顺序递增。
- **状态**：📋 待开发 | 🚧 开发中 | ✅ 已完成 | ⏸️ 暂停 | ❌ 已废弃
- **Commit**：关联的 git commit SHA（短格式）。
- **FR 编号**：对应 MVP_REQUIREMENTS.md 中的功能需求编号。
