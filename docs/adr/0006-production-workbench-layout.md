# ADR-0006：生产页采用专业工作台为主布局

- 状态：Accepted
- 日期：2026-07-16

## 背景

抛弃式原型比较了三种生产页结构：步骤向导、专业工作台和任务看板。生产阶段需要同时查看镜头结构、提示词、参考资产、结果历史和状态。

## 决策

- 前期创作流程继续使用顶部步骤向导。
- 即梦生产页以原型 B“专业工作台”为正式主结构：
  - 左侧镜头队列。
  - 中间视频/截图预览、结构化镜头字段与提示词。
  - 右侧参考资产、生成记录和失败反馈。
- 吸收原型 C 的状态筛选、数量统计和“下一个待处理镜头”能力。
- 不在正式产品保留 A/B/C 方案切换器。
- 正式页面使用动漫短剧示例与即梦术语。

## 结果

- 新用户依靠前期向导理解流程。
- 熟练用户在生产阶段获得较高信息密度和编辑效率。
- 页面实现可围绕稳定布局设计组件和 E2E 用例。
